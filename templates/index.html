DESCRIPTION

Our project HaySay is aimed to develop an application to expedite the monitoring of sheep and cattle breeding. In the system, the user will have various options to monitor. Firstly, mobility will be monitored thanks to the heat and pressure sensors that will be added to the animals. Pressure sensors were added to observe the rumination of the animals. The data from this sensor will be fed to our machine learning models in order to forecast the animals' health. We also added temperature sensors that will be assembled into animal ear tags. This value is suggested to be used in further research to monitor health. Additionally, a motion sensor is added to the system in order to facilitate the annual census, which is done manually every year. The users can do census by using motion sensors or uploading the animal ear tag to our system. Overall, this project is aimed to help farmers with monitoring the health of animals and annual/daily census easily.
